set.pl

 isSet(++Set:list) is semidet
True if Set does not contain repeated elements
 elementOf(?Element:any, ++Set:list) is nondet
True if Element belongs to Set. ?Element returns each set element.
 setDiff(++SetA:list, ++SetB:list, -Diff:list) is multi
True if Diff is the set difference SetA \ SetB, in a specific order.
 setUnion(++SetA:list, ++SetB:list, -Union:list) is det
True if Union is the set union SetA U SetB, in a specific order.
 subsetOf(?Subset:list, ++Set:list) is nondet
True if every element in Subset belongs to Set (without repeating). ?Subset returns every sequence of each subset of Set.
 equivalentTo(?SetA:list, ++SetB:list) is nondet
True if every element in SetA belongs to SetB (without repeating), with no one left. ?SetA returns every sequence of SetB.
 setIsOneOf(?Set:list, ++SetList:list) is nondet
True if Set is equivalent to any set in SetList. Equivalence is the the same defined by "equivalentTo". ?Set returns every sequence of each equivalent set in SetList.
 hasIntersection(+SetA:list, ++SetB:list) is semidet
True if SetA has a non-empty intersection with SetB.
